# Построение ионосферных карт по данным c gnss приемников.
Этот репозиторий содержит реализацию реконструкции карт общей электронной плотности (Total Electron Content, TEC) в ионосфере Земли с использованием сферических гармоник (Spherical Harmonics, SH).


Метод основан на разложении данных TEC по сферическим гармоникам для моделирования пространственно-временных вариаций ионосферы, что актуально в геофизике и навигационных системах GNSS.

- Реконструкция глобальных или региональных карт TEC из измерений (например, GNSS).

- Применение в исследованиях ионосферы, прогнозировании возмущений и коррекции сигналов.

- Подходит для анализа данных в области геофизики, связанных с вращением Земли и спектральным анализом



В папке report находится отчет с проделанной работой. Также, там можно посмотреть на мини-исследование: как изменится разрешение карт ПЭС, если приемников будет меньше (по сути, тоже самое, что и в обратную сторону. Насколько сильно изменится разрешение карты, если мы добавим новые приемники.)
